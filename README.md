# Bare-metal Rust Http Server.

Just an http server built with bare-metal Rust(Rust without a framework/library).

## How to use server

1. Clone this repository.
2. Install all the dependencies(and compile code-base).

```shell
cargo build
```

3. Run the server

```shell
cargo run
```

4. Update code-base to suit your use-case.

Cheers!!!
